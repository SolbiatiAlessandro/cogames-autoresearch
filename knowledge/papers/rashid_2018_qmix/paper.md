**Year / Venue:** ICML 2018 ŌĆö PMLR 80, pp. 4295ŌĆō4304; extended JMLR version 2020

**arXiv:** [1803.11485](https://arxiv.org/abs/1803.11485)

## Abstract

QMIX is a novel value-based method that can train decentralised policies in a centralised end-to-end fashion. It uses a network that estimates joint action-values as a complex non-linear combination of per-agent values that condition only on local observations. The method structurally enforces that the joint action-value is **monotonic** in the per-agent values, which allows tractable maximisation of the joint action-value in off-policy learning, and guarantees consistency between the centralised and decentralised policies. Testing on StarCraft II micromanagement tasks demonstrates that QMIX significantly outperforms existing value-based MARL methods.

## Introduction

Cooperative MARL faces a fundamental tension: agents must act using only local observations, yet coordination often requires knowledge of global state. The paradigm of **Centralized Training with Decentralized Execution (CTDE)** resolves this by allowing global state access during training while executing decentrally.

Prior approaches have limitations:
- **IQL:** Cannot represent inter-agent interactions; suffers from non-stationarity
- **VDN:** Factorises joint Q as additive sum of individual Q-values ŌĆö too restrictive, ignores global state
- **Fully Centralised (COMA):** On-policy learning; joint action space intractable

QMIX's key insight: full additive factorisation (VDN) is unnecessary. What's required is the **Individual-Global-Max (IGM)** principle ŌĆö that global argmax yields the same result as per-agent argmax. This is guaranteed by enforcing **monotonicity** rather than strict additivity.

## Methods

### Core Idea: Monotonicity Constraint

```
ŌłéQ_tot / ŌłéQ_a Ōēź 0,  for all agents a
```

This ensures that the decentralised greedy policy (each agent independently maximising Q_a) is globally consistent with the joint greedy policy over Q_tot. Joint maximisation is tractable in time **linear** in the number of agents.

### Architecture

**Agent Networks:** Each agent has a DRQN-based network for Q_a(Žä^a, u^a), conditioned on local action-observation history. Agents share network parameters.

**Mixing Network:** A feedforward network combining per-agent Q-values into Q_tot. **Weights (not biases) are restricted to be non-negative**, enforcing the monotonicity constraint. Used only during training.

**Hypernetworks:** The mixing network's weights and biases are generated by hypernetworks that take global state s as input:
- Weight hypernetworks use absolute value activation ŌåÆ non-negative outputs
- Bias hypernetworks are unconstrained

This allows Q_tot to flexibly depend on global state while preserving monotonicity.

### Training

Minimizes standard TD loss (DQN-style) with replay buffer and target network:

## Key Results

### Two-Step Matrix Game

VDN's additive factorisation cannot represent the optimal joint value function ŌåÆ converges to reward 7 (suboptimal). QMIX's non-linear monotonic mixing learns the optimal policy (reward 8).

### StarCraft II Micromanagement (6 maps)

| Method | Performance |
|---|---|
| IQL | Fails across all maps (high variance, performance collapse) |
| VDN | Improves over IQL but plateaus on complex heterogeneous scenarios |
| QMIX | Highest win rates across all maps; largest gains on heterogeneous agents (3s_5z, 1c_3s_5z) |

### Ablations

| Variant | Finding |
|---|---|
| QMIX-NS (no hypernetworks) | Performs on par with VDN; non-linearity alone without state doesn't help |
| QMIX-Lin (no hidden layer) | Underperforms full QMIX on heterogeneous maps |
| VDN-S (VDN + state term) | Fails to match QMIX; non-linear mixing is needed |

Both **non-linear mixing** and **state conditioning via hypernetworks** are jointly necessary.

### Learned Behaviors

On 2s_3z: VDN learns simple attack patterns; QMIX learns Zealots blocking enemies while Stalkers attack from range ŌĆö emergent tank-and-damage role differentiation.

---

## Conclusion

QMIX enables tractable off-policy learning of expressive joint action-value functions while guaranteeing globally consistent decentralised policies. By combining monotonic non-linear mixing with hypernetwork-based state conditioning, QMIX significantly outperforms both IQL and VDN.

QMIX has become a standard MARL baseline and inspired the StarCraft Multi-Agent Challenge (SMAC) benchmark and numerous follow-on methods (Weighted QMIX, QPLEX, etc.).
